部署模式Deployment Modes

本地部署Local Deployment

所有计算与数据处理在客户现场完成，数据不出场，断网可运行。适合对隐私要求严格的场景。All computation and data processing on-premises. Data never leaves, works offline. For high-privacy scenarios.

边云协同Edge-Cloud Synergy

核心推理在本地，非敏感数据与模型更新通过云端同步。兼顾隐私与持续进化。Core inference locally, non-sensitive data and model updates sync via cloud. Balances privacy and evolution.

私有化部署Private Cloud

在客户私有云或专属服务器上部署完整平台，适合大规模机构或集团化管理。Full platform on customer's private cloud or dedicated servers. For large-scale or group management.
