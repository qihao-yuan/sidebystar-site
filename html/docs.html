开发工具 Development Tools

事件总线 SDK Event Bus SDK

基于AsyncIO的高性能事件系统，支持发布/订阅模式、事件路由和优先级处理。 High-performance event system based on AsyncIO, supporting pub/sub pattern, event routing, and priority handling.

AsyncIO Pub/Sub 类型安全 Type Safety

使用示例: Usage Example: 

import asyncio
from aios.core.event_bus import Event, publish, BUS

# 订阅事件
async def handle_text(event: Event):
    print(f"收到文本: {event.data['text']}")

BUS.subscribe("stt/text", handle_text)

# 发布事件
await publish({
    "topic": "stt/text",
    "data": {"text": "你好", "confidence": 0.95},
    "source": "test_client"
})

装饰器示例: Decorator Example: 

from aios.utils import retry, timed, cache

@retry(retries=3, backoff=2.0)
@timed(name="API调用")
@cache(ttl=300)
async def api_call(url):
    # 带重试、计时和缓存的API调用
    response = await fetch(url)
    return response.json()

插件开发框架 Plugin Development Framework

基于规则的可扩展插件系统，支持设备控制、任务执行和动态加载。 Rule-based extensible plugin system supporting device control, task execution, and dynamic loading.

规则引擎Rule Engine 动态加载Dynamic Loading 插件管理Plugin Management

插件开发示例: Plugin Development Example: 

from aios.plugins import plugin_handler

@plugin_handler("control_light")
async def control_light(event):
    location = event.entities.get("location")
    action = event.entities.get("action")
    
    if action == "开启":
        return await turn_on_light(location)
    elif action == "关闭":
        return await turn_off_light(location)
