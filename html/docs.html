AIOS 技术概览 AIOS Technical Overview

循星科技AIOS (AI-Integrated Operating System) 是一个基于事件驱动架构的现代化AI居家服务平台。本手册详细介绍了AIOS的技术架构、开发工具和API接口。 SidebyStar AIOS (AI-Integrated Operating System) is a modern AI home service platform based on event-driven architecture. This manual provides detailed information about AIOS technical architecture, development tools, and API interfaces.

基于AsyncIO的高性能事件系统，支持发布/订阅模式、事件路由和优先级处理。 High-performance event system based on AsyncIO, supporting pub/sub pattern, event routing, and priority handling.

工具库API Utils Library API

44个标准化工具函数，涵盖路径操作、文件IO、时间处理、装饰器、并发控制等。 44 standardized utility functions covering path operations, file I/O, time processing, decorators, and concurrency control.

零依赖Zero Dependencies 类型提示Type Hints 跨平台Cross-Platform

基于规则的可扩展插件系统，支持设备控制、任务执行和动态加载。 Rule-based extensible plugin system supporting device control, task execution, and dynamic loading.

API 接口 API Interfaces

AIOS提供完整的RESTful API接口，包括环境感知系统、事件总线、工具库和插件系统的访问接口。 AIOS provides comprehensive RESTful API interfaces, including access to environmental sensing systems, event bus, utility libraries, and plugin systems.

  • 环境感知系统 API Environmental Sensing System API - 与环境感知硬件交互的标准接口 Standard interfaces for interacting with environmental sensing hardware
  • 事件总线 API - 异步事件系统核心API Event Bus API - Core API for asynchronous event system
  • 插件开发 API - 插件系统开发接口 Plugin Development API - Plugin system development interface
