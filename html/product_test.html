智能场景演示 Smart Scene Demos

场景一：宠物识别与看护 Scenario 1: Pet Recognition & Care

识别状态：Recognition Status: 狗狗在门前来回走动并吠叫 Dog pacing and barking at the door

  • 开启室内摄像头分析行为Activate indoor camera to analyze behavior
  • 播放主人声音的安抚语音Play owner's voice recording for comfort
  • 推送提醒给用户Send notification to user
  • 展示分析结果并询问是否需要进一步处理（例如：是否需要喂食喂水等）Display analysis results and ask for further action (e.g., need feeding/watering?)

已执行：摄像头已经开启，语音安抚中，结果已推送，展示进一步处理结果 Executed: Camera activated, comforting voice playing, notification sent, displaying further action options

场景二：起床自动化 Scenario 2: Wake-up Automation

检测动作：Detected Actions: 用户翻身 → 坐起 → 起身 User turns over → sits up → gets up

检测声音：Detected Sounds: 闹钟响起 → 用户起床 Alarm rings → User gets up

  • 判断室外亮度逐步打开窗帘Assess outdoor brightness and gradually open curtains
  • 调取天气api，获取天气信息Fetch weather info via weather API
  • 根据天气及传感器信息调整窗帘、灯光、空调等设备Adjust curtains, lights, AC based on weather and sensor data
  • 根据行为习惯推测下一步需求（例如：是否需要热水器预热，是否需要打开加湿器）Predict next needs based on habits (e.g., preheat water heater, turn on humidifier?)
  • 播放轻柔音乐Play gentle music

已执行：窗帘已打开，灯光已调高，春季干燥，热水器已预热，加湿器已打开，播放轻柔音乐 Executed: Curtains opened, lights brightened, dry season, water heater preheated, humidifier on, gentle music playing

场景三：睡眠体温监测与调节 Scenario 3: Sleep Temperature Monitoring & Adjustment

热成像状态：Thermal Imaging Status: 背部区域持续温度偏高 Back area consistently showing higher temperature

检测动作：Detected Actions: 踢掉被子/翻身 Kicked off blanket / Turned over

  • 降低空调目标温度至合适温度Lower AC target temperature to suitable level
  • 开启轻风模式Activate gentle breeze mode
  • 记录睡眠热图数据Record sleep thermal map data
  • 根据热图数据分析睡眠状态Analyze sleep state based on thermal map data

环境已调整，下一轮温控将在 30 分钟后进行，并根据热图数据综合分析睡眠状态 Environment adjusted, next temperature control in 30 mins, analyzing sleep state based on thermal data

场景四：语音查询交通状况 Scenario 4: Voice Query for Traffic Conditions

语音指令：Voice Command: "今天早上武林那边会堵吗？" "Will it be congested near Wulin this morning?"

意图解析：Intent Parsing: 查询 "武林"+"今天早高峰"+"拥堵状况"，出门可能 Query "Wulin" + "today morning peak" + "congestion status", likely for commute

调取地图api:Map API Call: "高德地图/百度地图" "AMap / Baidu Maps"

调取天气api:Weather API Call: "和风天气/中央气象台" "QWeather / CMA"

查询结果：Query Result: "武林广场车流密集，预计拥堵延迟约 12 分钟，今天有雨，湿度较大" "Heavy traffic near Wulin Square, estimated delay ~12 minutes. Rain today, high humidity."

本地语音播报:Local Voice Announcement: "武林广场方向建议提前出发，今天有雨，湿度较大，记得带伞" "Suggest leaving earlier for Wulin Square direction. It's raining today with high humidity, remember to take an umbrella."

播报完毕，实时追踪路况，并根据路况调整出行方案 Announcement finished. Tracking traffic in real-time and adjusting travel plans accordingly.
