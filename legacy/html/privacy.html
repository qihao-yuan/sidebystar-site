信任基础TRUST FOUNDATION

隐私优先，数据主权归用户Privacy First, Data Sovereignty for Users

循星科技以"数据本地化、算法端侧化、决策自治化"为核心原则，确保用户数据安全、可控、不出门。 SidebyStar follows core principles of data localization, edge-side algorithms, and autonomous decision-making to ensure user data remains secure, controlled, and local.

数据本地化Data Localization

  • 所有感知数据（视频、音频、传感器）默认在本地处理和存储，不上传至任何第三方云服务器All sensing data (video, audio, sensors) processed and stored locally by default, never uploaded to third-party cloud servers
  • 边缘 AI 推理完全在本地完成，模型运行不依赖网络连接Edge AI inference runs entirely locally, model execution does not depend on network connectivity
  • 断网状态下系统可正常运行，所有核心功能不受影响System operates normally when offline, all core functions remain unaffected

数据上传边界Data Upload Boundaries

默认不上传Not Uploaded by Default

视频流、音频数据、用户行为记录、个人偏好模型等敏感数据默认留在本地。Video streams, audio data, user behavior records, and personal preference models stay local by default.

可选同步Optional Sync

设备状态摘要、系统健康报告、脱敏统计数据可选择性云端同步，需用户/管理员明确授权。Device status summaries, system health reports, and anonymized statistics can optionally sync to cloud with explicit user/admin authorization.

模型更新Model Updates

模型升级包可通过安全通道下载，但本地训练数据不会回传。Model update packages can be downloaded via secure channels, but local training data is never sent back.

安全机制Security Mechanisms

  • 安全沙箱 -- 第三方插件与算法在隔离沙箱中运行，无法直接访问核心数据Security sandbox -- third-party plugins and algorithms run in isolated sandboxes without direct access to core data
  • 权限校验 -- 多角色权限体系（管理员/护理员/家属/维护人员），按需分配数据访问权限Permission verification -- multi-role permission system (admin/caregiver/family/maintenance) with need-based data access
  • 加密传输 -- 所有设备间通信采用加密协议，防止中间人攻击与数据截获Encrypted transmission -- all inter-device communication uses encrypted protocols to prevent MITM attacks and data interception
  • 日志审计 -- 所有系统操作与数据访问可追溯、可审计Audit logging -- all system operations and data access are traceable and auditable

视频与音频策略Video & Audio Policy

本地存储Local Storage

视频和音频数据仅在本地设备存储，保存周期可由用户/管理员配置。Video and audio stored only on local devices, retention period configurable by user/admin.

访问控制Access Control

视频/音频回放需通过身份认证与权限校验，支持多级审批。Video/audio playback requires identity verification and permission checks, supporting multi-level approval.

自动清理Auto Cleanup

超过保存周期的数据自动清理，不可恢复，确保无历史数据泄露风险。Data beyond retention period is automatically purged and unrecoverable, ensuring no historical data leakage risk.

数据归属声明Data Ownership Statement

  • 客户部署的系统中产生的所有数据，归属权归客户所有All data generated in customer-deployed systems belongs to the customer
  • 循星科技不会在未经授权的情况下访问、使用或分享客户数据SidebyStar will not access, use, or share customer data without authorization
  • 客户有权随时导出或删除系统中的所有数据Customers have the right to export or delete all data in the system at any time
